RESQ-AI 🚀 : "Turning Chaos into Coordination."
⚡ The Vision
When a disaster strikes, the first 60 minutes are the most critical. Currently, NGOs lose this "Golden Hour" to messy spreadsheets, 
fragmented WhatsApp groups, and information overload. RESQ-AI replaces the chaos with a unified, AI-powered nerve center.

## 🛠 Tech Stack
- **Frontend:** React + Tailwind CSS
- **Database/Real-time:** Firebase Firestore
- **AI:** Gemini 1.5 Pro (for automated crisis summarization)
- **Deployment:** Vercel

## 💡 The Problem
NGOs often face "bottlenecks" and fragmented communication during disasters. Manual data processing slows down life-saving resource allocation.

## ✨ Key Features
- **Real-time Coordination:** Live chat system with visual status indicators.
- **AI Intelligence:** Gemini-powered summaries of complex NGO datasets.
- **Interactive Dashboard:** Dynamic table for volunteer and resource matching.
